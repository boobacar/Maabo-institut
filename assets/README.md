# Assets Maabo Institut

Remplacez ces fichiers placeholders par vos vrais médias.

## Arborescence
- `images/logo/` → logo officiel (`logo-maabo.png` recommandé)
- `images/interieur/` → photos de l'institut (ex: `interieur-1.jpg`)
- `images/avant-apres/` → résultats clients (avant/après)
- `images/team/` → photos de l'équipe

## Conseils
- Format recommandé: JPG/WebP pour photos, PNG/SVG pour logo
- Largeur minimum: 1200px pour photos hero/galerie
- Compresser les images (TinyPNG/Squoosh) pour de meilleures performances
